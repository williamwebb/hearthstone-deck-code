============

hearthstone deck code encode/decoder

Usage
=====

```
// DeckCode.encode(deck: Deck)
// DeckCode.decode(deck: String)
```

Installation
=====
```
// Add the JitPack repo
repositories {
    maven { url 'https://jitpack.io' }
}

// Add dependency
dependencies {
    compile 'io.williamwebb:hearthstone-deck-code:<version>'
}
```
